# P2P Chat - Peer-to-Peer Messaging Application

![P2P Chat](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

A fully **Peer-to-Peer (P2P)** chat application that runs directly in the browser. Users can communicate with each other directly without any server.

## 🌐 Live Demo

[https://shofiqul5.github.io/dev/Random-communicate/](https://shofiqul5.github.io/dev/Random-communicate/)

## ✨ Features

### 💬 Messaging
- Text messaging
- Image sharing
- Video file sharing
- Audio file sharing
- Voice message recording

### 📞 Calling Features
- Audio call (beautiful UI with timer)
- Video call
- Call timer display
- Microphone mute/unmute
- Camera on/off
- Bandwidth monitoring
- Automatic quality adjustment

### 🎨 User Interface
- Mobile-friendly design
- Responsive layout
- Dark theme
- Touch-optimized buttons
- Username setup system

## 🚀 How to Use

### Online Usage
1. Open the website [https://shofiqul5.github.io/dev/Random-communicate/](Type)
2. Set your username (e.g., `rahim_123`)
3. Copy your username and share with friends
4. Connect using your friend's username
5. Start chatting!

### Offline/Local Usage
1. Download the complete folder
2. Open `index.html` file in your browser
3. Follow the steps above
