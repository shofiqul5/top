পরীক্ষণ নং - ২

HTML পেজে টেক্সট এর সাইজ নির্দিষ্ট করা এবং এ্যালাইন করা

তত্ত্ব: HTML ডকুমেন্টের পেজে লেখা টেক্সট এর সাইজ নির্দিষ্ট করা এবং এ্যালাইন করার জন্য font ট্যাগে এট্রিবিউট হিসাবে font align= এবং font size= এর পাশে এলাইনমেন্ট এবং সাইজ সংখ্যায় নির্দিষ্ট করা হয় ।

যন্ত্রপাতির ব্যবহার:

  1. হার্ডওয়ার: একটি কম্পিউটার
  2. সফটওয়ার: অপারেটিং সিস্টেম- Windows - 10
  3. এডিটিং সফটওয়্যার: যেমন- নোটপ্যাড ।
  4. ব্রাউজিং সফটওয়্যার: যেমন- গুগলক্রোম অথবা মজিলা ফায়ার ফক্স ।

প্রক্রিয়া অনুসরণ (কার্যপ্রণালী):

  1. কম্পিউটার চালু করে ডেস্কটপ এ আসি ।
  2. নোটপ্যাড ওপেন করে এতে HTML কোড লিখি । যেমন:
<html>
<head>
<title>Jhaudia College</title>
</head>
<body>
<p align="center"><b><font size="5">Welcome to Jhaudia College</font></b></p>
</body>
</html>

ফাইল সংরক্ষণ করা:

  1. File মেনুতে ক্লিক করে Save সাব মেনুতে ক্লিক করি । Save As ডায়ালগ বক্স প্রদর্শিত হবে ।
  2. File Name বক্সে যে কোন নাম লিখে .html লিখে Desktop সিলেক্ট করে Save বাটনে ক্লিক করে ডকুমেন্টটি সেভ করি । HTML ফাইলটি তৈরী হবে ।

ফলাফল:

Welcome to Jhaudia College

ব্যাখ্যা:

  • align="center" দ্বারা লেখাটি সেন্টার এ্যালাইন হওয়ার জন্য নির্দিষ্ট করা হয়েছে ।
  • <b> দ্বারা লেখাটি বোল্ড হওয়ার জন্য নির্দিষ্ট করা হয়েছে ।
  • font size="5" দ্বারা লেখাটির সাইজ নির্দিষ্ট করা হয়েছে ।

পরীক্ষণ নং - ৩

HTML পেজে হেডিং ট্যাগের ব্যবহার

তত্ত্ব: HTML ডকুমেন্টে হেডিং একটি গুরুত্বপূর্ণ বিষয়। হেডিংগুলো <h1> থেকে <h6> ট্যাগ দ্বারা নির্দিষ্ট করে দেওয়া হয়। <h1> ট্যাগ ব্যবহার করলে সবচেয়ে বড় হেডিং প্রদর্শিত হয়। আর <h6> ট্যাগ ব্যবহার করলে সবচেয়ে ছোট হেডিং প্রদর্শিত হয়।

যন্ত্রপাতির ব্যবহার:

  1. হার্ডওয়ার: একটি কম্পিউটার
  2. সফটওয়ার: অপারেটিং সিস্টেম- Windows - 10
  3. এডিটিং সফটওয়্যার: যেমন- নোটপ্যাড ।
  4. ব্রাউজিং সফটওয়্যার: যেমন- গুগলক্রোম অথবা মজিলা ফায়ার ফক্স ।

প্রক্রিয়া অনুসরণ (কার্যপ্রণালী):

  1. কম্পিউটার চালু করে ডেস্কটপ এ আসি ।
  2. নোটপ্যাড ওপেন করে এতে HTML কোড লিখি । যেমন:
<html>
<body>
<h1>This is heading 1</h1>
This is normal text (not heading)
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>
<h4>This is heading 4</h4>
<h5>This is heading 5</h5>
<h6>This is heading 6</h6>
</body>
</html>

ফাইল সংরক্ষণ করা:

  1. File মেনুতে ক্লিক করে Save সাব মেনুতে ক্লিক করি । Save As ডায়ালগ বক্স প্রদর্শিত হবে ।
  2. File Name বক্সে যে কোন নাম লিখে .html লিখে Desktop সিলেক্ট করে Save বাটনে ক্লিক করে ডকুমেন্টটি সেভ করি । HTML ফাইলটি তৈরী হবে ।

ফলাফল:

This is heading 1 This is normal text (not heading) This is heading 2 This is heading 3 This is heading 4 This is heading 5 This is heading 6

ব্যাখ্যা: উল্লিখিত পরীক্ষণটিতে যে সমস্ত ট্যাগ ব্যবহার করা হয়েছে:

ট্যাগসমূহবর্ণনা

<title> </title>ডকুমেন্টের টাইটেল নির্দেশ করে।
<body> </body>প্রোগ্রামের মূল কনটেন্ট অংশ নির্দেশ করে।
<h1> </h1>হেডার ট্যাগ ১ (সবচেয়ে বড়)
<h2> </h2>হেডার ট্যাগ ২

পরীক্ষণ নং - ৪

HTML এ টেক্সট ফরমেটিং এর ব্যবহার

তত্ত্ব: আউটপুটকে ফরমেটিং করার জন্য HTML এ বিভিন্ন ধরনের ফরমেটিং রয়েছে। ফরমেটিং বলতে মূলত টেক্সট ফরমেটিংকেই বুঝানো হয়। HTML এ লেখা Bold, Italic, Underline, Subscript, Superscript ইত্যাদি টেক্সট ফরমেটিং এর জন্য <b>, <i>, <u>, <sub>, <sup> ইত্যাদি ট্যাগ ব্যবহৃত হয়।

যন্ত্রপাতির ব্যবহার:

  1. হার্ডওয়ার: একটি কম্পিউটার
  2. সফটওয়ার: অপারেটিং সিস্টেম- Windows - 10
  3. এডিটিং সফটওয়্যার: যেমন- নোটপ্যাড ।
  4. ব্রাউজিং সফটওয়্যার: যেমন- গুগলক্রোম অথবা মজিলা ফায়ার ফক্স ।

প্রক্রিয়া অনুসরণ (কার্যপ্রণালী):

  1. কম্পিউটার চালু করে ডেস্কটপ এ আসি ।
  2. নোটপ্যাড ওপেন করে এতে HTML কোড লিখি । যেমন:
<html>
<body>
<p><b>This text is bold</b></p>
<p><u>This text is underline</u></p>
<p><i>This text is italic</i></p>
<p><small>This text is small</small></p>
<p>This is<sub> subscript</sub> and <sup>superscript</sup></p>
</body>
</html>

ফাইল সংরক্ষণ করা:

  1. File মেনুতে ক্লিক করে Save সাব মেনুতে ক্লিক করি । Save As ডায়ালগ বক্স প্রদর্শিত হবে ।
  2. File Name বক্সে যে কোন নাম লিখে .html লিখে Desktop সিলেক্ট করে Save বাটনে ক্লিক করে ডকুমেন্টটি সেভ করি । HTML ফাইলটি তৈরী হবে ।

ফলাফল:

This text is bold This text is underline This text is italic This text is small This is subscript and superscript

ব্যাখ্যা:

  • প্রতিটি লাইনকে <p> </p> ট্যাগের ভেতর রেখে আলাদা প্যারাগ্রাফ হিসেবে এগুলো প্রথমে নির্ধারণ করা হয়েছে ।
  • বোল্ডের জন্য ব্যবহার করা হয়েছে - <b> </b> ট্যাগ ।
  • আন্ডারলাইনের জন্য ব্যবহার করা হয়েছে - <u> </u> ট্যাগ ।
  • ইটালিকের জন্য ব্যবহার করা হয়েছে - <i> </i> ট্যাগ ।
  • স্মল লেটারের জন্য ব্যবহার করা হয়েছে - <small> </small> ট্যাগ ।
  • সাবস্ক্রিপ্ট এর জন্য ব্যবহার করা হয়েছে - <sub> </sub> ট্যাগ ।
  • সুপারস্ক্রিপ্ট এর জন্য ব্যবহার করা হয়েছে - <sup> </sup> ট্যাগ ।

পরীক্ষণ নং - ৬

Hello World প্রিন্ট করার প্রোগ্রাম (C Programming)

প্রয়োজনীয় যন্ত্রপাতি: একটি কম্পিউটার ।

প্রয়োজনীয় সফটওয়্যার:

  1. একটি অপারেটিং সিস্টেম । যেমন: Windows - 10
  2. একটি প্রোগ্রামিং ভাষা । যেমন: Dev-C++ Compiler.

যন্ত্রপাতির ব্যবহার:

  • কাজের শুরুতে কম্পিউটারের বৈদ্যুতিক সংযোগ ঠিক আছে কি না তা দেখে নিতে হবে ।
  • বৈদ্যুতিক সংযোগ ঠিক থাকলে কম্পিউটারের পাওয়ার সুইচ অন করে কম্পিউটারটি চালু করতে হবে ।
  • বর্ণিত পরীক্ষণটি সম্পন্ন করার জন্য প্রয়োজনীয় প্রোগ্রামটি চালু করতে হবে ।

প্রক্রিয়া অনুসরণ:

  1. Dev-C++ এর উপর ক্লিক করে প্রোগ্রামটি চালু করতে হবে ।
  2. File ক্লিক → New ক্লিক → Source File নির্বাচন করি ।
  3. কোড এডিটরে প্রয়োজনীয় কোড লিখতে হবে ।
#include<stdio.h>
int main()
{
    printf("Hello World!");
    return 0;
}

ফাইল সংরক্ষণ করা: File ক্লিক → Save ক্লিক → File Name বক্সে যে কোন নাম লিখে .c লিখতে হবে । → Desktop সিলেক্ট করে → Save বাটনে ক্লিক করতে হবে । ফাইলটি নির্ধারিত লোকেশনে সেভ হবে ।

কম্পাইলার ও রান করা: সেভ করা ফাইলটি ওপেন করতে হবে । কম্পাইলার ও রান করার জন্য Execute ক্লিক করে → Compile & Run ক্লিক করতে হবে ।

ফলাফল:

Hello World! Process exited after 0.04581 seconds with return value 0 press any key to continue ---

ব্যাখ্যা: সি প্রোগ্রামে কোন আউটপুট পাওয়ার জন্য printf() ফাংশন ব্যবহার করা হয় । তাই এই প্রোগ্রামে printf() ফাংশনের মধ্যে Hello World! লেখা হয়েছে । আউটপুট স্ক্রিনে সেটাই প্রিন্ট হয়েছে ।

পরীক্ষণ নং - ৭

একটি নির্দিষ্ট টেক্সটকে একাধিকবার প্রদর্শনের প্রোগ্রাম (C Programming)

প্রয়োজনীয় যন্ত্রপাতি: একটি কম্পিউটার।

প্রয়োজনীয় সফটওয়্যার:

  1. একটি অপারেটিং সিস্টেম। যেমন: Windows - 10
  2. একটি প্রোগ্রামিং ভাষা। যেমন: Dev-C++ Compiler.

যন্ত্রপাতির ব্যবহার:

  • কাজের শুরুতে কম্পিউটারের বৈদ্যুতিক সংযোগ ঠিক আছে কি না তা দেখে নিতে হবে।
  • বৈদ্যুতিক সংযোগ ঠিক থাকলে কম্পিউটারের পাওয়ার সুইচ অন করে কম্পিউটারটি চালু করতে হবে।
  • বর্ণিত পরীক্ষণটি সম্পন্ন করার জন্য প্রয়োজনীয় প্রোগ্রামটি চালু করতে হবে।

প্রক্রিয়া অনুসরণ:

  1. Dev-C++ এর উপর ক্লিক করে প্রোগ্রামটি চালু করতে হবে।
  2. File ক্লিক → New ক্লিক → Source File ক্লিক করে নির্বাচন করি।
  3. কোড এডিটরে প্রয়োজনীয় কোড লিখতে হবে।
#include<stdio.h>
int main()
{
    int i;
    i = 0;
    while(i < 5)
    {
        printf("I Love Bangladesh. \n");
        i = i + 1;
    }
    return 0;
}

ফাইল সংরক্ষণ করা: File ক্লিক → Save ক্লিক → File Name বক্সে যে কোন নাম লিখে .c লিখতে হবে। → Desktop সিলেক্ট করে → Save বাটনে ক্লিক করতে হবে। ফাইলটি নির্ধারিত লোকেশনে সেভ হবে।

কম্পাইলার ও রান করা: সেভ করা ফাইলটি ওপেন করতে হবে। কম্পাইলার ও রান করার জন্য Execute ক্লিক করে → Compile & Run ক্লিক করতে হবে।

ফলাফল:

I Love Bangladesh. I Love Bangladesh. I Love Bangladesh. I Love Bangladesh. I Love Bangladesh. Process exited after 0.04966 seconds with return value 0 press any key to continue---

ব্যাখ্যা:

  • #include<stdio.h> লাইনটি প্রতিটি প্রোগ্রামের প্রথম লাইন হিসেবে বসাতে হয়। stdio অর্থ standard input output.
  • main() একটি ফাংশন। যখন প্রোগ্রাম রান শুরু করা হয় তখন main() ফাংশনটি প্রথমেই নির্বাহিত হয়।
  • { হলো শুরু বন্ধনী এবং } হলো শেষ বন্ধনী।
